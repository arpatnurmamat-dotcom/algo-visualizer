课程表问题（拓扑排序）

中等

📝 题目描述

你这个学期必须选修 numCourses 门课程，记为 0 到 numCourses - 1。

在选修某些课程之前需要一些先修课程。先修课程按数组 prerequisites 给出，其中 prerequisites[i] = [ai, bi]，表示如果要学习课程 ai 则必须先学习课程 bi。

请你判断是否可能完成所有课程的学习？如果可以，返回 true；否则，返回 false。

💡 思路解析

拓扑排序的核心思想是：在有向无环图（DAG）中，不断找到入度为 0 的节点并移除，若最终所有节点都被处理，则说明无环；否则存在环。

本题采用 BFS（Kahn 算法）实现：

1. 构建邻接表，计算每个节点的入度。

2. 将所有入度为 0 的节点加入队列。

3. 依次从队列取出节点，将其所有邻居的入度减 1，若邻居入度变为 0 则加入队列。

4. 若最终处理的节点数等于总课程数，则无环，可完成全部课程。

📊 复杂度分析

时间复杂度: O(V + E)  |  空间复杂度: O(V + E)

其中 V 为课程数，E 为先修关系数。每个节点和每条边最多被访问一次。
